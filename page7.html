USING EAGLE

Eagle is a PCB modelling software which can be used to configure the wiring of PCB circuits and mill PCB boards.

STEP 1

The first step of using eagle is to import a library of all the relevant components you will be using in your PCB.

These components can then be found in the library section of the left tab in Eagle.

STEP 2

Having started a new library, select file and then schematic to start a new circuit diagram

STEP 3

To add components to the schematic, press edit, and then add to select from library
