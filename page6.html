MILLING A CIRCUIT BOARD

Stock PCBs can be custom milled to allow for many different types of circuits to be constructed from the same initial component. Before milling a circuit board, the circuit has to be designed on a CAD program. These can be downloaded and seen below. 

DETAIL OUTLINE

LOADING A MILLED PCB

STEP 1

Locate the relevant components and order them so you know what they are when it comes to loading them onto the board

STEP 2

Get some solder ready and turn on the soldering iron.

STEP 3

The soldering process is as follows:

Tin (apply some solder) to the joining surface on the PCB

Using tweezers, position one end of the component you want to solder over the relevant contact it needs to be joined to

Ensuring the component contact is touching the tinned connection, heat this area by touching it gently with the soldering iron.

This should create a weak join, and allow enough stability for you to solder the next connection point more thoroughly, do this

Once this point has been soldered properly, return to the first connection point and ensure that the join is solid. 

Repeat this process for each component.


STEP 4

Once all components have been attached, the PCB pin connections need to be tinned to ensure that they have enough contact with a USB port. This photo shows untinned PCB connections, they should be a consistent silver colour when properly tinned.
